General

ATES(Automation Test Eco System) is a website designed for test automation execution for WebDriver(Selenium 2) test cases written in JAVA.


The system enables you to:

  • Execute test cases on slaves with different browsers any time, anywhere
  • Troubleshoot test failures with detailed test reports
  • Oversee the automation coverage with historical execution results

Integration Requirements

To make full use of the system, we need to make sure following items are ready:

  • The automation code is written in JAVA
  • The automation code is annotated by TestNG
  • The source is integrated with MAVEN, and it's preferred that the project source structure follows MAVEN recommendations.

FAQ

Can we run cases with JUnit annotations?

No.

TestNG is the only framework supported so far. Please make sure the annotations are converted to TestNG. You can refer to this link for more information.


How do we pass in testng.xml parameters to our test case?

For example, we have a testng.xml defined as below:

<suite name="testng.xml sample">
    <parameter name="browser" value="FireFox" />
    <parameter name="environment" value="UAT" />
    <parameter name="config" value="properties/config.properties" />

    <test name="Smoke Test" preserve-order="true">
        <groups>
            <run>
                <include name="smoke" />
            </run>
        </groups>
        <packages>
            <package name="com.*" />
        </packages>
    </test>
</suite>

There are three parameters needed to be passed: browser=FireFox, environment=UAT and config=properties/config.properties.

In this case, two actions are needed to make the job done. First, make a minor change to the JAVA code references those parameters. For example, change code from: 

@Parameters({ "browser", "environment", "config" })
public void initConfig(String browser, String environment, String config) {
    Properties prop = null;
    prop = initializeProperties(config);
    prop.setProperty("browser", browser);
    prop.setProperty("environment", environment);
}

To:

@Parameters({ "browser", "environment", "config" })
public void initConfig(String browser, String environment, String config) {
+   if (browser == null || browser.isEmpty()) {
+       browser = System.getProperty("browser");
+   }
+   if (environment == null || environment.isEmpty()) {
+       environment = System.getProperty("environment");
+   }
+   if (config == null || config.isEmpty()) {
+       config = System.getProperty("config");
+   }
    Properties prop = null;
    prop = initializeProperties(config);
    prop.setProperty("browser", browser);
    prop.setProperty("environment", environment);
}
