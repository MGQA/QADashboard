General

ATES(Automation Test Echo System) is a website designed for test automation execution for WebDriver(Selenium 2) test cases written in JAVA.


The system enables you to:

  • Execute test cases on slaves with different browsers any time, anywhere
  • Trouble shoot test failures with detailed test reports
  • Oversee the automation coverage with historical execution results
